# 숫자야구

모든 클래스는 main으로 불러와서 동작시킨다.

# 기능 정의

### Target 숫자 구현

- computer 클래스 생성
    - makeTarget메소스 생성
        - `camp.nextstep.edu.missionutils.Randoms`의 `pickNumberInRange()`

### 입력 구현

- user 클래스 생성
- 입력 메소드
    - `camp.nextstep.edu.missionutils.Console`의 `readLine()`
- 중복된 입력 에러 구현
### 심판 구현

- umpire 클래스 생성
- 스트라이크 체크
    - 같은 수가 같은 자리에 있으면 스트라이크
- 볼 체크
    - 다른 자리에 있으면 볼
- 낫싱 체크
    - 같은 수가 전혀 없으면 낫싱
  
### 리팩토링
 - 숫자야구 3자리 변경 가능하도록 상수처리
 - 전체 코드 컨벤션 확인
